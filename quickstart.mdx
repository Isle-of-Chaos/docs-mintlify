## Web

Go to https://qrbtf.com/ai


### 1: Enter a URL

Enter or Scan the corresponding URL for the QR code you need to generate.

<img src="/images/quickstart/URL.jpg" style={{ width: '400px', borderRadius: '0.5rem' }} />

### 2: Enter the Prompt

Enter a description of the image you want to create in the prompt field or generate prompts randomly.

<img src="/images/quickstart/prompt.jpg" style={{ width: '400px', borderRadius: '0.5rem' }} />

### 3: Enter the API key

Enter **qrbtf-ai** in the API key field.
