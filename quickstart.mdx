## Web

Go to https://qrbtf.com/ai


### 1: Type a URL

Type or Scan the corresponding URL for the QR code you need to generate.

<img src="/images/quickstart/URL.jpg" style={{ width: '400px', borderRadius: '0.5rem' }} />

### 2: Type the Prompt

Type a description of the image you want to create in the prompt field or generate prompts randomly.

<img src="/images/quickstart/prompt.jpg" style={{ width: '400px', borderRadius: '0.5rem' }} />

### 3: Type a API key

Type **qrbtf-ai** in the API key field. 

<img src="/images/quickstart/API.jpg" style={{ width: '400px', borderRadius: '0.5rem' }} />
